WangDesk

Quick Start
1. Extract this zip to any folder.
2. Double-click WangDesk.App.exe.
3. If no main window appears, check the system tray near the clock.

Notes
- This package is self-contained. You do not need to install .NET.
- Settings are stored in %AppData%\WangDesk\settings.json.
- To remove WangDesk, close it from the tray and delete this folder.

快速开始
1. 把压缩包解压到任意目录。
2. 双击 WangDesk.App.exe。
3. 如果没有看到主窗口，请到右下角系统托盘查看图标。

说明
- 这是自包含版本，不需要额外安装 .NET。
- 设置文件保存在 %AppData%\WangDesk\settings.json。
- 如需删除 WangDesk，请先从托盘退出程序，再删除整个文件夹。
