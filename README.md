# WangDesk

[English](README.md) | [简体中文](README.zh-CN.md)

A Windows system tray desktop pet with Pomodoro timer and quick tools.

WangDesk is a lightweight Windows productivity app that lives in the **system tray** and shows an animated **desktop pet**. It combines a **Pomodoro timer**, quick device status checks, and practical utilities for focused work sessions.

## Why WangDesk

- Keep focus with an always-ready Pomodoro workflow.
- Monitor key system metrics (CPU, memory, disk) from tray UI.
- Get a playful desktop companion that reflects current device activity.

## Features

- Animated desktop pet in the Windows system tray.
- Pomodoro timer window for focus and break sessions.
- Quick system status panel for CPU, memory, and disk usage.
- Tray-based quick tools and settings.
- Optional auto-start on Windows boot.

## Project Description

Windows system tray desktop pet with Pomodoro timer, device status and quick tools.

## Target Platform

- Windows 10 / Windows 11

## Development Requirements

- .NET 9 SDK

## Quick Start

### For regular Windows users (recommended)

1. Use `setup.exe` generated by `Build-Setup.cmd`, or download the installer from a release.
2. Double-click `setup.exe`.
3. Follow the installer wizard to finish installation.
4. Launch WangDesk from the Start menu or desktop shortcut.
5. No .NET installation is required for this installer.

### Portable package

1. Use the packaged build `WangDesk-win-x64.zip` generated by `Package-WangDesk.cmd`, or download the same zip from a release.
2. Extract the zip to any folder.
3. Double-click `WangDesk.App.exe`.
4. If no main window appears, check the WangDesk icon in the Windows system tray near the clock.

### From source code

1. Double-click `Start-WangDesk.cmd` in the project root.
2. If WangDesk is already built, it will open immediately.
3. On first launch, the script will build the app automatically when `.NET 9 SDK` is installed.

### For developers

```bash
dotnet restore
dotnet run --project src/WangDesk.App/WangDesk.App.csproj
```

## Build

```bash
dotnet build src/WangDesk.App/WangDesk.App.csproj
```

## Publish

Double-click `Build-Setup.cmd` to create a Windows installer for end users.

Installer files:

- `artifacts\installers\WangDesk-Setup.exe`
- `artifacts\installers\setup.exe`

Double-click `Package-WangDesk.cmd` to create a portable self-contained package.

Output files:

- `artifacts\packages\WangDesk-win-x64\`
- `artifacts\packages\WangDesk-win-x64.zip`

## Configuration

Settings file location:

`%AppData%\\WangDesk\\settings.json`
